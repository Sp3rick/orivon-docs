# Dashboard

Orivon implements it's own [Dashboard Application](../orivon#applications--also-called-special-extensions-) which is going to be the default one

Dashboards Apps has access to Details, Icons, and Widgets from other Apps or Web3sites and has permissions such as opening links, interacting with widgets and interacting with some settings

The Orivon Dashboard is going to be Android inspired, with a resizable grid where Apps and Widgets can be placed into, some pre-inserted Apps may include App stores, featured Web3sites, Wallet management, Network, Nodes, Settings etc...  
Mouse scrolling will change dashboard page up to left and down to right
